# SunnyCreativeLog
創ることを、少しずつ積み重ねる。創作活動の記録と振り返りのための、個人用Webアプリです。

## ReleaseLog
- 2026-07-01 v0.1 Sunrise
  - 作品と文字数を記録できるようになった
- 2026-07-03 v0.2 Bloom
  - 作品とログを分離
  - HTML/CSS/JavaScriptを分離
  - localStorageの構造を変更
- 2026-07-04 v0.3 Notebook
  - 作品選択UI追加
  - バグ修正
    - 無題作品の作成を防止
    - 日付を日本時間で判定するよう修正
    - 空のログを登録できないよう修正
    - 同じ作品が重複して作成される不具合を修正
  - 作品一覧を選択中の種別で絞り込む機能追加
      
## 現在の機能
- 創作ログ登録
- 作品ごとの管理
- 今日の文字数集計
- localStorage保存
